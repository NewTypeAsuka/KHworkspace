배열

        Java 배열: 같은 자료형의 변수를 묶음으로 다루는 것
        JS 배열: 변수를 묶음으로 다루는 것

        JS 변수의 특징: 값이 저장되기 전까지는 자료형을 알 수 없음
        -> 값이 없으면 자료형이 지정되지 않음

        JS 배열의 특징
        1) 자료형 제한이 없음
        2) 길이 제한이 없음

        [JS 배열 선언 방법]
        const arr = new Array(); // 0칸 짜리 빈 배열 생성
        const arr = new Array(3); // 3칸 짜리 배열 생성
        const arr = []; // 0칸 짜리 빈 배열 생성
        const arr = [1, 2, 3]; // 배열 생성 및 초기화

배열 관련 함수

        - 배열 관련 함수는 prototype이 Array인 배열에만 사용이 가능하다.
        - Object, NodeList, HTMLCollection에는 사용 불가
        
        (Stack 구조 관련 함수)
        배열명.push(): 배열 마지막 요소로 추가
        배열명.pop(): 배열 마지막 요소를 꺼내옴
        배열명.unshift(): 배열 첫번째 요소로 추가
        배열명.shift(): 배열 첫번째 요소를 꺼내옴
        배열명.indexOf("값"): 배열 내 요소 중 "값"이 일치하는 요소의 index 반환(없으면 -1 반환)
        배열명.sort([정렬기준함수]): 배열을 정렬 기준 함수에 맞춰 정렬(정렬 기준 함수 미작성 시 문자열 순서로 정렬)
        배열명.toString(): 모든 요소를 하나의 문자열로 반환(요소 사이에 , 추가)
        배열명.join("구분자"): 모든 요소를 하나의 문자열로 반환(요소 사이에 구분자 추가)
        배열명.forEach(function(value){}):
        배열 요소를 하나씩 순서대로 꺼내서 value에 저장한 다음 function 내부 코드를 순서대로 수행
